QueryResponse

public struct QueryResponse

A structure to hold a response from a query operation

  • records

    A list of records matching the provided query. If the includeData flag was not set in the query, these records will contain empty RecordData values.

    An identifier for the final result of the query. Use this value in a subsequent query by setting the after property of a QueryParams object.
