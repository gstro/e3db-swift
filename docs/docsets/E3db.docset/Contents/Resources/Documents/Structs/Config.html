  • init(clientName:clientId:apiKeyId:apiSecret:publicKey:privateKey:baseApiUrl:)

    Initializer to customize the configuration of the client. Typically, library users will use the Client.register(token:clientName:apiUrl:completion:) method which will supply an initialized Config object. Use this initializer if you register with the other registration method, Client.register(token:clientName:publicKey:apiUrl:completion:). Pass this object to the Client.init(config:) initializer to create a new Client.
